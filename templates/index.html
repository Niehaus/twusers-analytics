Bem vindo! Neste site iremos analisar algumas informações do seu Twitter, mas para isso precisamos de sua permissão. Não se preocupe, antes disso vamos te explicar o que vai acontecer.

Para começar, precisamos de dados do seu Twitter, não se preocupe isso só é possível com a sua permissão. Mais informações foram disponibilizadas pelo próprio Twitter em OAuth.

Aqui uma listinha do que vai acontecer:

  1. We're going to generate an OAuth request token that we will use when sending you to Twitter to grant us access.
  2. Then we will redirect you to Twitter.com using our tokens from step 1, for you to authorize our app.
  3. While at Twitter.com you will authorize twauth-web and you'll then be redirected back to us.
  4. We will consume the OAuth params passed back to our callback URL. Using those values, we will then get your auth token from the Twitter API.
  5. Finally, to confirm that all has been authorized, we'll make an authenticated call to the Twitter API to get some information about the user who authorized the app, to show we are able to retrieve this data on your behalf.
